# **App Name**: Nexus Portfolio

## Core Features:

- Hero Section Presentation: Displays an engaging introduction with title, personal subtitle, descriptive text, and a call-to-action, alongside a minimalistic illustration in a dual-column layout for larger screens.
- Skills Dashboard Display: Renders a skill categorization interface (Software, Expertise, Language) with a dynamic grid of compact skill cards, each featuring an icon and name, defaulting to 'Software'.
- Projects Showcase: Presents categorized projects ('Featured', 'All Projects') within a responsive grid of detailed project cards, including image/icon, title, description, and category, defaulting to 'Featured'.
- Collaborate Call-to-Action: Displays a direct contact section encouraging collaboration, featuring a persuasive paragraph and prominent email and WhatsApp buttons with direct links.
- Responsive Layout Adaptation: Ensures the entire portfolio gracefully adapts its structure and elements across various screen sizes and orientations for optimal viewing and a consistent user experience.
- Interactive Visual Feedback: Implements subtle, smooth hover animations, including slight color changes for skill cards and elevation for project cards and buttons, enhancing interactivity without being intrusive.

## Style Guidelines:

- Primary Color: #9C9EE8 (Muted Indigo) - A modern, desaturated indigo providing a subtle, professional accent for interactive elements and key highlights.
- Background Color: #1B1B1F (Very Dark Indigo Grey) - A deeply desaturated, almost black background with a hint of cool indigo, adhering to the requested 'negro/gris muy oscuro' for a sophisticated dark mode.
- Accent Color: #B2DBED (Pale Azure) - An analogous hue to the primary, offering clear yet restrained visual contrast for specific elements, maintaining the overall professional and minimalist feel.
- Main Text Color: #FFFFFF (White) - Utilized for primary content to ensure maximum readability and impact against the dark background.
- Secondary Text Color: #A0A0A0 (Mid-Grey) - Provides hierarchical distinction and a softened appearance for less critical information, improving readability in dark mode.
- Card Background Color: #2B2B2F (Dark Grey) - Slightly lighter than the main background, this deep grey creates visual depth and clearly defines UI components like skill and project cards.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and objective aesthetic, suitable for all textual content.
- Use minimalist line-art or monochrome icons for illustrative elements in the hero section and descriptive visuals for skill sets and project categories, ensuring a cohesive, professional look.
- The main content will be centered within a single-column layout, with a maximum width of approximately 900px, ensuring focus and readability. Sections will be clearly separated by ample vertical spacing.
- The Hero section will feature a dual-column layout on desktop, presenting text on the left and a minimalist illustration on the right. Skill cards will be displayed in a 3-column grid on desktop, while project cards will use a 2-column grid.
- Subtle, smooth animations will be applied on hover states for interactive elements like buttons, skill cards, and project cards, including slight color transitions and elevation effects, to enhance user engagement without being distracting.
